# build-base

[![Build](https://github.com/gsmlg-ci/build-base/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/build-base/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/build-base`
- `ghcr.io/gsmlg-dev/build-base`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/build-base:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/build-base:latest
```

## Build

```bash
docker build -t gsmlg/build-base .
```
